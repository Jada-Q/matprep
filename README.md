# MATPREP

> 8 周在家自主练巴西柔术（BJJ）的单页手册 · 零基础友好

**在线使用：** https://matprep-ten.vercel.app

## 是什么

一个纯前端单文件 app（`index.html`），做上垫前的体能与动作准备。
**不替代道馆真人指导**——招式实战必须在教练在场下学。第 8 周的终点是去道馆上第一节体验课。

## 功能

- **练前自检** — 今天能不能练的 6 项安全红线 + 身体状态
- **供能** — 按体重实时算蛋白 / 碳水 / 饮水 / 热量区间
- **8 周计划** — 周一至周日训练排布 + 进度勾选
- **动作图鉴** — 13 个 solo 动作，简笔图 + 编号步骤 + 视频
- **训练模式** — 全屏跟练，语音报动作 + 计时，趴垫子上不用碰屏
- **打印卡** — ⌘P 出一页今日训练卡贴墙
- **习惯系统** — 连续天数 🔥 + 8 周热图 + 手动打勾
- **动物爬行** — 6 个趣味移动热身（熊爬 / 蟹走 / 鳄鱼爬…）
- **训练日志** — 日 / 周 / 月复盘
- **分享** — 发给朋友，各自进度独立

## 技术

- 单文件 `index.html`，零依赖，纯静态
- 进度存浏览器 localStorage（本机，不上传任何服务器）
- streak 连续天数逻辑有单元测试：`node streak.test.mjs`（8/8 通过）

## 安全红线

颈部只做等长 · 受身严格渐进 · 每次必热身 5 分钟 · 频率 > 强度
